## Run Locally

Clone the project

```bash
  git clone git@github.com: aketchgithub/codeChallenge1.git
```

Go to the project directory

```bash
  cd codeChallenge
```

Install dependencies

```bash
  npm install
   "prompt-sync": "^4.2.0"
```

Start the server

```bash
  node main.js
```


## Running Tests

To run tests, run the following command

```bash
  npm run test
```
